Playground

Hier kannst du üben. Gebe deinen Code in das Textfeld ein, verwende jshero.log zur Ausgabe und führe den Code mit 'Run' aus. Zum Beispiel: 

var hallo = function() {
  return "Hallo Welt!";
};

jshero.log("Der Aufruf 'hallo()' ergibt:");
jshero.log(hallo());
Probiere es aus! Kopiere das Beispiel in das Textfeld und führe den Code aus.
